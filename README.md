## Getting Started

First, run the development server:

```bash
pnpm i
pnpm dev

# 배포를 위해 firebase-tools 설치(처음 1회만)
npm i -g firebase-tools

firebase login

# 배포
pnpm run deploy
```

Open [http://localhost:5173/](http://localhost:5173/) with your browser to see the result.
